🔷 Google Drive Setup Guide

Yeh ek baar karna hai — phir hamesha ke liye Google Drive se data sync hoga!
Aap duniya mein kahin se bhi apna data access kar sakeinge. 🌍
📋 Step by Step:
1
Google Cloud Console kholein
console.cloud.google.com par jayein aur apne Google account se login karein.
2
Naya Project banayein
Top mein "Select Project" → "New Project" → Naam likhein jaise MySchoolApp → Create
3
Google Drive API Enable karein
Left menu → "APIs & Services" → "Library" → Search: Google Drive API → Enable
4
OAuth Consent Screen set karein
"APIs & Services" → "OAuth consent screen" → External → App name likhein → Save
5
Credentials banayein
"APIs & Services" → "Credentials" → "Create Credentials" → "OAuth Client ID" → Application type: Web Application
6
Authorized Origins add karein
"Authorized JavaScript origins" mein add karein:
https://aapki-site.github.io
↑ Yeh aapke hosting ka URL hona chahiye
7
Client ID copy karein
Create ke baad ek Client ID milega jaise: 123456-abc.apps.googleusercontent.com
Ise copy karein aur Settings → Google Drive Client ID mein paste karein.
8
Free Hosting (Agar local file hai)
Agar aap HTML file se chal rahe hain to pehle GitHub Pages par host karein:
github.com → New repo → Upload HTML file → Settings → Pages → Enable
URL milega: username.github.io/repo-name
✅ Koi payment nahi! Google Cloud free tier kaafi hai — school data ke liye bilkul free.
